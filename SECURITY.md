## Responsible Disclosure

I kindly ask that you follow responsible disclosure practices:
- Do not publicly share or exploit the vulnerability before it is fixed  
- Allow reasonable time for investigation and patching  
- Avoid conducting destructive testing on production environments

Thank you for helping improve the security of this project.
